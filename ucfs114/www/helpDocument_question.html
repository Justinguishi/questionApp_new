Welcome to questionApp


The questionApp is designed to upload and insert a new question to a database on the web server. Users can create a new question by clicking on a point on the leaflet map and add the question.



>To start the questionApp, first of all, please launch the browser. Next, enter the URL (e.g. http://developer.cege.ucl.ac.uk:30314/) Then, you will be led to the home page of the App.(If you are out of the UCL, please connect to the UCL VPN first!)
The layout of the app: The main section consists of a scalable map provided by OpenStreetMap.The + and - signs in the upper left corner of the map can zoom in and out. The entire page contains several other functional buttons, "get Earthquakes", "trackAndCircle" and so on.
(When you launch the App, the port number will be showed in an alert, here, my port number=30314）


>You can set the question for a location by filling the question form below the map and submitting your question to the database.The lat/lng of the question point can be automatically entered the form when the you clicks on a map in the question app. In addition, if you doesn’t type in a number between 1-4 for the correct answer field or if the you doesn’t fill in all the required fields, an error message would show.

>Besides these core functions, you can find some other functions on the both left-hand side and right-hand side menu bar.

1.getPort function will let you know your port number(eg. mine is 30314)


2.Load quiz point will show all the quiz points on the map.


3.add PointLinePoly will build a point a circle and a triangle in the central London area on map.


4.Get Earthquakes function will load the earthquake data provided, showing them on the map.


5.track Location function will offer the client's location and also calculate the distance from UCL campus, if the distance is within 1 km, alert would show up.


6.UCL Logo function(on the right-hand side menu bar) will load the UCL logo on the top of the map
